# Node.js Microservices Task Manager

A microservices-based backend application built using Node.js and Express.

## Services

### API Gateway
- Routes requests to services

### Auth Service
- User Registration
- User Login
- JWT Authentication

### Task Service
- Create Task
- Get Tasks
- Update Task
- Delete Task

### Notification Service
- Notification APIs

## Tech Stack

- Node.js
- Express.js
- MariaDB
- JWT
- Microservices Architecture
- API Gateway

## Run

### Auth Service

```bash
npm install
npm run dev
```

### Task Service

```bash
npm install
npm run dev
```

### API Gateway

```bash
npm install
npm run dev
```
